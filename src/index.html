Personal projects

My way of learning is getting hands dirty. The computer is an excellent teacher as it allows no mistakes, but it lets you be who you are.

EBM ^3.1.0

The Easy Box Model is a npm package that allows very rapid and modular prototyping setting the foundations for what can become a bigger product.

It is modular by itself as it can fit in any web framework by assigning a source and destination path.

It also introduces a modular Sass and modular JS file structure that meets the MVC model directory organization of most web applications.

Elang API

Can computers create their own opinion?

Elang API parses human generated content and classifies it in multiple categories such as POS, context, mood.

Elang API began as a scraper for words in spanish in order to have a wide and structured spanish words database.

This project is always a WIP because languages are always evolving too.

ECOMMERCE UI

A Laravel ^5.4 e-commerce CMS web application that allows users to list their own products and administrators to track orders from an admin-panel dashboard built with the EBM called ANOKIODA (Another Kind Of Dashboard).

This personal project is a response for the complexity of working with other e-commerce frameworks out there.

It is also an intent of applying the EBM modular principles to a real-world project.
