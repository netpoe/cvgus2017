Proyecto Tés Arome

El objetivo es desarrollar un theme de Shopify que esté listo para recibir integraciones de e-commerce y al menos un Google Tag Manager para poder tagearlo

Estructura Referencias Presupuesto

Estructura

La estructura del sitio comprende:

  • Homepage (header)
    • Nuestros Tés - CTA (Agregar al carrito, Comprar)
    • Accesorios para té - CTA (Agregar al carrito, Comprar)
    • Conoce el mundo del té
    • En dónde comprar Tés Aromes
    • ¿Quiénes somos?
  • Contacto

Referencias

Tés Arome debe tener una estructura parecida a Kusmitea pero con un estilo depurado y que refleja el lujo.

Presupuesto y tiempos de entrega

El proyecto deberá finalizarse a finales de Noviembre 2017 ya habiendo revisado el contenido y el funcionamiento del mismo.

* Para que el cliente pueda recibir el código fuente del proyecto, éste deberá ser pagado en su totalidad.

* El cliente deberá de proveer los assets de diseño y contenido como logotipos, imágenes, paleta de colores y descripción de las secciones.
